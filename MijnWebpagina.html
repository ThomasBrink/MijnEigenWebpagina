Mijn eigen webpagina

Thomas Brink

Ik heb voor de opleiding software development gekozen omdat het me leuk leek om software te kunnen maken. Hiervoor had ik nog geen evaring met software developen. Wat er voorzorgt dat ik gemotiveerd blijf is voornamelijk dat ik het leuk vind om te expirimenteren met de dingen die ik heb geleerd

Mijn ontwikkelings punten zijn dat ik graag zonder terug te kijken naar eerder gemaakte codes een code kan gaan schrijven en dat css makelijk kan gaan toepassen met bijvoorbeeld het floaten van images. Voor dat ik aan deze opleiding begon heb ik mavo gedaan op het stedelijk dalton lyceeum. Ik heb nog geen bijbaan gehad

Bij mijn voor opleiding heb ik niet echt software developer gerelateerde kennis op gedaan behalve dan ik een profielwerkstuk moest maken over een vervolg opleiding en ik denk zelf dat misschien wiskunde bruikbaar kan zijn aangezien je daar voor logisch moet denken.
